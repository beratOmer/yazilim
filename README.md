Giriş
Yazılım ve donanım, bilgisayar sonuçlarının iki temel bileşenidir.

Donanım → Uluslararası parçalar

Yazılım → Donanımı yönetilen komutlar

Birbirleri olmadan çalışamazlar.

Donanım (Hardware) Nedir?
Elle tutulabilen tüm parçalardır.

Temel Donanım Kompleri

RAM → Geçici bellek

HDD / SSD → kalıcı depolama

GPU → Grafik İşlemleri

Anakart → Tüm parçaların ayrılması

PSU → Güç kaynağı
